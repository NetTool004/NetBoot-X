# NetBoot-X

Windows utility for boot and network optimization with automated tweaks and a dashboard interface.

## Features
- Automated boot optimizations
- Network performance enhancements
- System restore functions
- Interactive dashboard for configuration

## Quick Start
1. Run `NetBootX_Setup.iss` to install the program
2. Follow the dashboard instructions to apply tweaks
3. Restart your PC after major boot or network tweaks

## File Overview
- `main.py` - Main program entry point
- `dashboard.py` - Dashboard UI
- `boot_tweaks.py` - Boot optimization scripts
- `net_tweaks.py` - Network tweak scripts
- `restore.py` - System restore utilities
- `settings.json` / `state.json` - Configuration files
- `NetBootX_Setup.iss` - Inno Setup installer script
- `README.md` - Project description
